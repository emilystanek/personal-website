# Interactive Portfolio Website ✨

A unique, interactive portfolio website that stands out from the crowd. Built with Next.js, React, and Tailwind CSS with creative & thoughtful design.

## 🎨 Unique Features

### Interactive Elements
- **Interactive Typography Hero** ✨ - Each letter bounces and rotates on hover
  - Space Grotesk font for a tech-forward, unique look
  - **Cosmic Gradient subtitle** - Smooth animated gradient flowing through indigo → purple → pink like a nebula
  - Clean centered layout
  - Gradient underline on title
  - Night sky effect with twinkling stars scattered across the background
  - **30 wandering stars** - All stars disappear and reappear in different locations
  - Stars vary in size (2-4px) with double-layer glow effects
  - Each star moves to a unique position (45-120px away from origin)
  - 30-second animation cycles with staggered delays
  - Smooth fade in/out animations for magical effect
  - Scroll indicator at bottom
- **Custom MySpace-Style Cursor** 🌟 - Nostalgic custom cursor with trailing sparkles and stars (just like the old internet days!)
  - Glowing purple cursor ring that grows on hover
  - Gradient trail effect following your mouse
  - Random sparkle ✨ particles that appear as you move
  - Perfect blend of nostalgia and modern design
- **Glowing Technology Badges** ✨ - Each tech badge glows with its own unique color
  - Matching glow effects for each technology
  - Glow intensifies on hover
  - Smooth transitions and animations
- **Hover Animations** - Cards that lift, rotate, and transform on hover with 3D effects
- **Interactive Typography** - Each letter in the name bounces and rotates when you hover over it
- **Magnetic Buttons** - Social links that scale and lift when hovered
- **Animated Skill Tags** - Technology badges with gradient colors that rotate and scale on hover
- **Scroll Parallax** - Hero section with parallax scrolling effect
- **Smooth Reveals** - Timeline entries fade and slide in with staggered animations

### Creative Design
- **Bento Box Layout** - Modern card-based design system
- **Split Layouts** - Asymmetric compositions that guide the eye
- **Gradient Accents** - Colorful gradient borders and backgrounds throughout
- **Custom Scrollbar** - Branded purple gradient scrollbar
- **Thoughtful Typography** - Clear hierarchy and readable text
- **Playful Icons** - Emojis used thoughtfully for visual interest
- **Color Psychology** - Indigo/purple scheme for creativity and innovation
- **Micro-interactions** - Delightful details in every hover and click

### Performance
- ⚡ Next.js 16 with optimal bundle size
- 🚀 Zero heavy animation libraries (pure CSS + React)
- 🎯 GPU-accelerated transforms
- 📦 Lightweight and fast

## 🚀 Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser and experience the interactions!

## ✏️ Customization

Edit `app/page.tsx` to personalize:
- Your name and title in the hero section
- Replace the placeholder visual with your photo or custom illustration
- About section cards
- Technology skills and colors
- Work timeline
- Evolution journey items
- Social media links

### Color Schemes
The site uses a purple/indigo gradient theme. To change colors, search for:
- `indigo-` - Primary brand color
- `purple-` - Secondary accent
- `slate-` - Neutral grays
